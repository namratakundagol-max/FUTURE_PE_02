# FUTURE_PE_02 – AI Content Marketing using UGC Ads

## 📌 Overview
This project demonstrates AI-driven content marketing by generating a complete UGC Ad Content Pack for the FitLife Studio 90-Day Transformation Program in Bangalore.

## 🎯 Objective
To create scroll-stopping, conversion-focused UGC ad scripts designed to increase free trial bookings and program sign-ups.

## 📦 Deliverables
- 7 Scroll-stopping hooks
- 3 UGC-style video scripts
- Platform-specific ad adaptations
- 5 high-converting captions
- 5 strong CTA variations

## 🛠 Tools Used
- ChatGPT
- Google Docs
- GitHub

## 🧠 Skills Demonstrated
- UGC scriptwriting
- Ad psychology & hook creation
- Conversion-focused messaging
- Prompt engineering
- Multi-platform adaptation
